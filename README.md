# Pixel Study Buddy – Pomodoro Desktop App

A cozy, customizable Pomodoro timer built with **Electron.js** that helps you stay focused while studying or working.
Designed to feel like a lightweight desktop widget with themes, fonts, and motivational messages.

---

## ✨ Features

* ⏱ **Pomodoro Timer**

  * Work / Short Break / Long Break cycle
  * Automatic session transitions

* 🎨 **Customizable Themes**

  * Pastel
  * Sakura
  * Galaxy
  * Cyberpunk
  * Matcha
  * Sunset
  * Night mode

* 🔤 **Custom Fonts**

  * Pixel
  * Retro terminal
  * Typewriter
  * Cute modern UI

* 📊 **Daily Productivity Stats**

  * Pomodoros completed today
  * Total focus minutes tracked

* 💬 **Motivational Messages**

  * Random focus and break messages to keep you going

* 🪟 **Mini Mode**

  * Shrinks into a small floating widget
  * Perfect for keeping the timer visible while working

* 🔔 **Sound Feedback**

  * Session completion notification
  * Soft click interactions

* 🖼 **Custom Background GIF**

  * Add your own cozy or motivational GIF

## 🎯 Why I Built This

Most lightweight Pomodoro timers are web-based and run inside a browser tab.
When the tab is closed, the browser sleeps in the background, or you switch tasks, the timer can pause or stop entirely.

While studying on the same laptop, I often found that browser-based timers were unreliable and distracting.

So I built **Pixel Study Buddy**, a lightweight desktop Pomodoro timer that:

* stays active independently of browser tabs
* runs as a small floating widget while working
* provides a cozy and customizable study environment

The goal was to create a **minimal, distraction-free focus companion** that stays with you throughout your work session.

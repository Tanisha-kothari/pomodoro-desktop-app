# Pixel Study Buddy – Pomodoro Desktop App

A cozy, customizable Pomodoro timer built with **Electron.js** that helps you stay focused while studying or working.
Designed to feel like a lightweight desktop widget with themes, fonts, and motivational messages.

---

## ✨ Features

* ⏱ **Pomodoro Timer**

  * Work / Short Break / Long Break cycle
  * Automatic session transitions

* 🎨 **Customizable Themes**

  * Pastel
  * Sakura
  * Galaxy
  * Cyberpunk
  * Matcha
  * Sunset
  * Night mode

* 🔤 **Custom Fonts**

  * Pixel
  * Retro terminal
  * Typewriter
  * Cute modern UI

* 📊 **Daily Productivity Stats**

  * Pomodoros completed today
  * Total focus minutes tracked

* 💬 **Motivational Messages**

  * Random focus and break messages to keep you going

* 🪟 **Mini Mode**

  * Shrinks into a small floating widget
  * Perfect for keeping the timer visible while working

* 🔔 **Sound Feedback**

  * Session completion notification
  * Soft click interactions

* 🖼 **Custom Background GIF**

  * Add your own cozy or motivational GIF

## 🚀 Installation

### Option 1 – Download Release

Download the `.dmg` or `.exe` from the **Releases** section.

### Option 2 – Run from Source

```bash
git clone https://github.com/Tanisha-kothari/Pomodoro-desktop-app.git
cd Pomodoro-desktop-app
npm install
npm start
```

---

## 🛠 Build the App

```bash
npm run build
```

This generates installers inside:

```
dist/
```

---

## 🎯 Why I Built This

Most Pomodoro timers feel boring or cluttered.
Pixel Study Buddy was built to create a **cozy, customizable study companion** that feels pleasant to keep on your desktop while focusing.
